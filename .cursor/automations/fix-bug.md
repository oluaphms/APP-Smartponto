# Bug Fix System

## Objetivo
Corrigir o problema SEM quebrar outras partes do sistema.

## Passos
1. Usar análise anterior
2. Corrigir apenas o necessário
3. Explicar a correção
4. Mostrar código antes/depois
5. Validar impacto no sistema

## Regras
- Não refatorar tudo
- Não inventar código desnecessário
- Manter padrão existente
- Evitar efeitos colaterais

## Validação obrigatória
- Isso afeta login?
- Isso afeta estado global?
- Isso pode gerar loop?
