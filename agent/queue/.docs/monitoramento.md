# Monitoramento

## Objetivo
Visualizar eventos e status de marcações em tempo real.

## Antes de começar
- Confirme o período e os filtros desejados.

## Passo a passo
1. Acesse **Monitoramento**.
2. Aplique **Unidade** `Matriz` e **Setor** `Operações`.
3. Observe o painel de **alertas**.
4. Abra detalhes para ver marcações recentes.

## Checklist
- [ ] Filtros aplicados
- [ ] Alertas revisados

## Erros comuns
- Monitorar sem filtros e perder eventos relevantes
