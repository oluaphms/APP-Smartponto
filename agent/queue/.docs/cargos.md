# Cargos

## Objetivo
Manter a lista de cargos para cadastro e relatórios.

## Antes de começar
- Revise o organograma para evitar duplicidades.

## Passo a passo
1. Acesse **Cargos**.
2. Clique em **Novo**.
3. Preencha **Nome** `Analista Financeiro` e **Descrição** `Responsável por contas a pagar`.
4. Clique em **Salvar**.
5. Para atualizar, selecione e clique em **Editar**.
6. Para desativar, use **Inativar**.

## Checklist
- [ ] Nome padronizado
- [ ] Descrição clara (opcional)

## Erros comuns
- Cargo duplicado com grafia diferente
