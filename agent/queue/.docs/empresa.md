# Empresa

## Objetivo
Atualizar dados cadastrais e parâmetros gerais.

## Antes de começar
- Tenha em mãos o CNPJ e dados atualizados de contato.

## Passo a passo
1. Acesse **Empresa**.
2. Preencha **Razão Social** `PontoWebDesk LTDA`, **CNPJ** `12.345.678/0001-90` e **Telefone** `(11) 4000-0000`.
3. Atualize o endereço.
4. Clique em **Salvar**.
