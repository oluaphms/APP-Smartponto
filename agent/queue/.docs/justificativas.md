# Justificativas

## Objetivo
Manter motivos padronizados para ajustes e ocorrências.

## Antes de começar
- Defina categorias padrão (ex.: Atestado, Ajuste, Banco de Horas).

## Passo a passo
1. Acesse **Justificativas**.
2. Clique em **Novo**.
3. Preencha **Nome** `Atestado médico` e **Categoria** `Afastamento`.
4. Clique em **Salvar**.
5. Para atualizar, selecione e **Edite**.

## Checklist
- [ ] Nome padronizado
- [ ] Categoria correta

## Erros comuns
- Criar motivos duplicados
