# Estruturas

## Objetivo
Configurar a estrutura organizacional para filtros e permissões.

## Antes de começar
- Mapeie níveis: Unidade, Setor, Centro de Custo.

## Passo a passo
1. Acesse **Estruturas**.
2. Clique em **Novo**.
3. Preencha **Nome** `Unidade São Paulo`, **Tipo** `Unidade` e **Vínculo** `Matriz` (se aplicável).
4. Clique em **Salvar**.
5. Repita para os demais níveis.
6. Revise a hierarquia e ajuste vínculos.

## Checklist
- [ ] Hierarquia definida
- [ ] Vínculos corretos

## Erros comuns
- Níveis duplicados
- Vínculos incorretos entre níveis
