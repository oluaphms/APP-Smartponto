# Fiscalização REP-P

## Objetivo
Gerar dados e evidências para fiscalização do REP-P.

## Antes de começar
- Verifique se os equipamentos estão sincronizados.

## Passo a passo
1. Acesse **Fiscalização REP-P**.
2. Selecione **Período** `01/04/2026 a 30/04/2026` e **Equipamentos** `REP A`.
3. Clique em **Gerar**.
4. Baixe o arquivo/relatório gerado.

## Checklist
- [ ] Período definido
- [ ] Equipamentos selecionados

## Erros comuns
- Gerar sem sincronizar equipamentos
