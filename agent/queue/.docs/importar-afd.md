# Importar AFD

## Objetivo
Importar arquivos AFD e atualizar registros de ponto.

## Antes de começar
- Garanta que o AFD pertence ao período correto.

## Passo a passo
1. Acesse **Importar AFD**.
2. Selecione o **equipamento** (ex.: `REP A`).
3. Escolha o **arquivo AFD** (ex.: `AFD_abril_2026.txt`).
4. Clique em **Importar**.
5. Aguarde o processamento e revise o resumo.

## Checklist
- [ ] Arquivo correto
- [ ] Período conferido

## Erros comuns
- Importar arquivo de período errado
