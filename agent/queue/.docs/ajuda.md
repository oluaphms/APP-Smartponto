# Ajuda

## Objetivo
Centralizar as instruções de uso de todas as páginas para apoiar o RH.

## Antes de começar
- Tenha em mãos o erro ou dúvida específica.

## Passo a passo
1. Acesse **Ajuda**.
2. Localize a página desejada na lista de conteúdos.
3. Abra o guia correspondente e siga o passo a passo.
4. Se necessário, acione o **suporte**.
