# Configurações

## Objetivo
Definir parâmetros globais do sistema.

## Antes de começar
- Verifique quais integrações estão habilitadas.

## Passo a passo
1. Acesse **Configurações**.
2. Ajuste **Idioma** `pt-BR` e **Notificações** `Ativas`.
3. Revise integrações (ex.: e-mail, webhooks).
4. Clique em **Salvar**.

## Checklist
- [ ] Idioma definido
- [ ] Notificações revisadas

## Erros comuns
- Alterar integração sem testar
